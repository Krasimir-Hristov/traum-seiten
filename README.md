# Traumseiten - AI Children's Stories SaaS

An AI-powered SaaS platform that allows users to create personalized children's stories with magical, pixel-perfect user interfaces, custom avatars, and safe, robust AI model integrations.

## Architecture
- **Frontend**: Next.js (App Router), Tailwind CSS (v4), TypeScript.
- **Backend**: FastAPI, Python, Pydantic, Supabase Auth.
- **AI Backend**: OpenRouter API for centralized LLM access.

## Project Structure
- `/frontend`: Next.js frontend application.
- `/backend`: FastAPI backend application.
- `api-contract.md`: The single source of truth for all API definitions.

## Workflow Rules
All development in this project adheres to strict coding standards, feature-based modules, and verifiable single-source-of-truth configuration files. See the `.agent` folder for full details.
